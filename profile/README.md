## 為什麼香港機房這麼受中國大陸用戶歡迎？

先把背景說清楚。

香港機房的核心優勢是地理位置。物理距離近，三網（電信、聯通、移動）到香港的延遲普遍在 30～50ms 之間，跑一個 `ping` 測試，能感受到它和其他海外節點的差距有多大。

另外，香港不需要像大陸那樣做 ICP 備案，這對很多需要快速上線業務、跨境電商、外貿建站的人來說，省掉了很多手續和等待時間。

香港 VPS 的線路質量，從低到高大致是：普通國際線路 → BGP 直連 → CMI / CN2 精品網絡 → CN2 GIA 頂級線路。不同線路對三網用戶的體驗差異很大，這一點後面在場景裡會具體說。
